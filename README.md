# FGC for Jamaica

A website dedicated to supporting Jamaica's recovery from Hurricane Beryl, created by Team Panama FGC.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Features

- **Home**: Overview of the situation in Jamaica
- **Support**: Ways to help and donate (including UN link placeholder)
- **About**: Detailed information about the situation and Team Jamaica FGC
- **FGC**: Video showcase of support messages from FGC teams worldwide

## Tech Stack

- React 18
- Vite
- React Router DOM
